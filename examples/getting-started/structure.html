Structure

Presentations contain "sections" and sections contain "slides". Here is an example of a presentation with multiple sections and slides:

slides:
  Section 1:
    - <h1>Introduction</h1>
    - <h1>Background</h1>
  Section 2:
    - <h1>Theory</h1>
    - <h1>Practice</h1>
  • Note, a slide can be:
    • raw HTML (as you see above)
    • the name of a file containing HTML
    • a YAML file listing a collection of slides
